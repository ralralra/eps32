# 2단계 — 마름/촉촉 판단하기

## 목표
1단계에서 정한 기준값으로 ESP32가 **스스로 "말랐다/촉촉하다"를 판단**하게 한다.

## 원리 — if 문 (조건 판단)
컴퓨터가 결정을 내리는 가장 기본 도구가 `if` 예요.
```cpp
if (raw > DRY_THRESHOLD) {
  // 기준값보다 크면 = 말랐다
} else {
  // 아니면 = 촉촉하다
}
```
센서 값 하나(숫자)를 **기준과 비교**해서 행동을 나누는 것. 이게 자동화의 핵심이에요.

## 해보기
1. `step2_judge_dry.ino` 열기
2. **`DRY_THRESHOLD` 값을 1단계에서 적어둔 내 기준값으로** 수정
   ```cpp
   int DRY_THRESHOLD = 3000;   // ← 내 센서 값에 맞게
   ```
3. 업로드 → 시리얼 모니터
4. 센서를 마른 곳/젖은 곳에 옮기며 메시지가 바뀌는지 확인

## 확인 ✅
- [ ] 마른 흙(또는 공기 중) → "🏜 흙이 말랐어요!"
- [ ] 젖은 흙(또는 물컵) → "💧 촉촉해요"
- [ ] 손으로 센서를 적셨다 말렸다 하면 메시지가 바뀐다

## 기준값이 안 맞으면?
- 항상 "말랐어요"만 나옴 → `DRY_THRESHOLD` 를 더 **크게**
- 항상 "촉촉해요"만 나옴 → `DRY_THRESHOLD` 를 더 **작게**
- 1단계의 공기중/물컵 값 사이로 조정

## 배운 것
- `if / else` 로 조건에 따라 다르게 행동한다
- 임계값(기준)을 바꾸면 판단이 달라진다 → 민감도 조절
- 아직은 "말로만" 알려줌 → 다음 단계에서 진짜 물 주기
