# 🌱 스마트팜 프로젝트 — Wemos D1 R32 + Apps Script + AI Studio 앱

> 출석체크·교실 환경 모니터링에서 배운 IoT 구조를 그대로, 이번엔 **식물을 돌보는 시스템**으로!
> 아두이노 키트의 센서·장치를 쓰되 보드는 **Wemos D1 R32 (ESP32)** — WiFi가 내장이라 별도 통신 모듈이 필요 없어요.
> 앱은 앱인벤터 대신 **AI Studio(Build → New App)** 로 만들어 폰에 설치(PWA)합니다.

## 시스템 전체 구조

```
🌱 화분/재배기
   │  토양습도 · 온습도 · 조도 · 수위 …        ← 입력(센서)
   ▼
📡 Wemos D1 R32 (ESP32)
   │  ① 10초마다 측정값 업로드                 ② 명령 폴링 (10초마다 "할 일 있어?")
   ▼                                           ▲
🧾 Apps Script 웹앱 (/exec) ←──────────────────┘
   │  doGet: 기록 저장 · 명령 전달 · 최신값 응답
   ▼
📊 구글 시트  ←  기록(자동 쌓임) + 설정 탭(명령·기준값)
   ▲
📱 AI Studio 앱 (폰 홈 화면에 설치)
      센서값 확인 · 펌프/팬/조명 ON·OFF · 자동모드 기준값 변경
```

- **측정 흐름**: 센서 → ESP32 → `URL?soil=…&temp=…` → 시트에 한 줄씩 기록 → 앱이 `?mode=latest`로 읽어 표시
- **제어 흐름**: 앱 버튼 → `?mode=set&cmd=PUMP_ON` → 시트 설정!A1 → ESP32가 `?mode=cmd`로 읽어 실행
- 교실 모니터링(4회차)과 **완전히 같은 구조**예요. 달라지는 건 센서 종류와 "출력장치(펌프·팬)"뿐!

## 단계 로드맵

| 단계 | 내용 | 확인 방법 |
|:---:|---|---|
| 1 | **센서 점검** — 키트의 센서를 하나씩 시리얼로 읽기 | 시리얼 모니터(115200) |
| 2 | **출력 점검** — 릴레이(펌프)·팬·조명을 코드로 ON/OFF | 눈으로! |
| 3 | **자동 제어** — 토양습도가 기준보다 마르면 펌프, 더우면 팬 | 시리얼 + 눈 |
| 4 | **클라우드 연결** — 측정값 시트 저장 + 원격 명령 | 구글 시트 |
| 5 | **AI Studio 앱** — 폰에서 확인·제어 | 폰! |

- 4단계 준비: [`01_docs/apps_script_setup.md`](01_docs/apps_script_setup.md)
- 5단계 준비: [`01_docs/ai_studio_prompt.md`](01_docs/ai_studio_prompt.md)

## ⚠ 아두이노 키트를 D1 R32에서 쓸 때 규칙 3가지

1. **핀 번호는 변환해서!** 키트 강의자료의 핀 번호는 우노 기준이에요. D1 R32에 꽂을 땐 변환표를 보세요 →
   [우노 라벨 ↔ GPIO 변환표](../esp32-class-projects/01_docs/pinmap_wemos_d1_r32.md)
   (예: 강의자료의 `D6` → 코드에는 `27`)
2. **아날로그 센서는 A2~A5 위치(GPIO 35·34·36·39)에!** — A0·A1 자리는 WiFi를 켜면 못 읽어요. 값 범위도 우노 0~1023이 아니라 **0~4095**입니다.
3. **모터·펌프·팬은 보드 핀에 직접 연결 금지** — 반드시 **릴레이 모듈**(또는 모터 드라이버)을 거치고, 펌프 전원은 보드가 아니라 **외부 5V**(USB 어댑터·건전지)에서 받으세요. 신호선만 ESP32에 연결!

## 기본 구성 (일반적인 스마트팜 키트 기준 — PDF 확인 후 확정)

| 역할 | 부품 | 연결 | 코드에서 |
|---|---|---|---|
| 흙 마름 감지 | 토양습도(수분) 센서 | A2 자리 (GPIO35) | `analogRead(35)` |
| 공기 온습도 | DHT11 | 디지털 (예: GPIO27) | DHT 라이브러리 |
| 밝기 | 조도(CDS) 센서 | A3 자리 (GPIO34) | `analogRead(34)` |
| 물탱크 잔량 | 수위 센서 | A4 자리 (GPIO36, 입력 전용 OK) | `analogRead(36)` |
| 급수 | 릴레이 + 미니 워터펌프 | 릴레이 IN → GPIO26 | `digitalWrite(26, HIGH)` |
| 환기 | DC 팬 (릴레이 경유) | 릴레이 IN → GPIO25 | `digitalWrite(25, HIGH)` |
| 보광(식물등) | LED 모듈 | GPIO17 | `digitalWrite(17, HIGH)` |

## 있으면 더 좋은 부품 추천

| 부품 | 왜 좋은가 | 난이도 |
|---|---|:---:|
| **수위 센서** (키트에 없다면 꼭!) | 물탱크가 비면 펌프가 공회전하다 망가져요 — "물 부족" 경보의 핵심 | ★ |
| **서보 모터 (SG90)** | 창문/차광막 개폐 데모 — "온도 높음 → 창문 열기" 같은 눈에 보이는 자동화 | ★★ |
| **네오픽셀 LED 스트립** | 식물등 흉내 + 상태 표시등(초록=정상, 빨강=물 부족) 겸용 | ★★ |
| **DS18B20 방수 온도센서** | 물탱크·흙 속 온도 측정 — 공기 온도와 비교하는 탐구 활동 | ★★ |
| **부저** | 물 부족·고온 경보음 (앱 알림과 이중화) | ★ |

> pH·EC(양액 농도) 센서는 비싸고 보정이 어려워 수업용으론 비추천이에요.
