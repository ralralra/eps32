# 3단계 — 자동 급수 (릴레이 + 펌프)

## 목표
흙이 마르면 ESP32가 **릴레이로 펌프를 켜서 진짜 물을 주게** 한다. 화분 완성의 핵심 단계!

## ⚠ 먼저 — 안전 규칙 (꼭!)
**펌프 전원은 외부 5V에서. ESP32나 USB에서 직접 X.**
펌프가 전류를 많이 먹어서 ESP32가 Brownout으로 꺼져요(출석 때 겪은 그 현상). ESP32는 **릴레이라는 스위치만** 조작합니다.

```
[ESP32 GPIO26] → 릴레이 IN (신호만)
[외부 5V] → 릴레이 → 펌프   (실제 전력)
```
배선 자세히는 [`../docs/wiring_and_setup.md`](../docs/wiring_and_setup.md)

## 원리 — 릴레이 = 전자 스위치
릴레이는 ESP32의 작은 신호로 **큰 전류(펌프)를 켰다 껐다** 하는 스위치예요.
```cpp
digitalWrite(RELAY_PIN, PUMP_ON);   // 펌프 켜기
delay(2000);                         // 2초 동안
digitalWrite(RELAY_PIN, PUMP_OFF);  // 펌프 끄기
```

## 해보기
1. `DRY_THRESHOLD` 를 내 센서 기준값으로 수정
2. 펌프 튜브를 물컵에 담그고, 출구는 화분(또는 빈 컵)으로
3. 업로드 → 시리얼 모니터로 동작 관찰
4. 센서를 마른 상태로 두면 → "급수 시작" 뜨고 펌프가 돈다
5. 센서를 적시면 → "촉촉함" 뜨고 급수 안 함

## 확인 ✅
- [ ] 마르면 펌프가 2초 돌고 멈춘다
- [ ] 급수 후 5초 대기했다가 다시 측정한다
- [ ] 촉촉하면 펌프가 안 돈다
- [ ] 시리얼 메시지와 펌프 동작이 일치한다

## 잘 안 될 때
- **펌프가 계속 켜져 있거나 반대로 동작** → 릴레이가 active-HIGH 타입. 코드에서
  `PUMP_ON`/`PUMP_OFF` 의 `LOW`/`HIGH` 를 서로 바꾸기
- **ESP32가 자꾸 리셋(Brownout)** → 펌프를 ESP32 전원에서 끌어다 쓴 것. 외부 5V로 분리
- **펌프가 약하거나 안 돎** → 외부 전원 전압/전류 확인, 튜브 막힘 확인

## 직접 바꿔보기
- 급수 시간(`WATER_MS`)·대기(`SOAK_MS`)를 조절
- 너무 자주 주면 과습 → 대기 시간을 길게

## 배운 것
- 릴레이로 큰 전력 장치를 안전하게 제어한다
- 센서 → 판단 → **장치 작동(액추에이터)** 까지 완성 = 자동화의 완성형
- 급수 후 대기(스며들 시간)처럼 **현실을 고려한 타이밍**이 중요
