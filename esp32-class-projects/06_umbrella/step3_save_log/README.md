# 3단계 — 대여 상태/기록 저장

## 목표
전원을 껐다 켜도 **"어떤 우산을 누가 빌려갔는지"가 그대로 남아있게** 한다. + 대여/반납 내역 기록.

## 원리 — 상태를 파일에 저장하고, 부팅 때 불러오기
2단계의 `borrowedBy[]`(누가 빌렸나)는 전원을 끄면 사라졌어요.
이번엔 그걸 파일에 저장하고, 켜질 때 다시 읽어옵니다.
```cpp
saveState();   // 상태가 바뀔 때마다 파일에 저장
loadState();   // 부팅 시 파일에서 복원
```
이게 진짜 시스템의 핵심이에요 — **꺼져도 잊지 않는 것.**

## 해보기
1. UID들을 내 카드값으로 수정 (2단계와 동일)
2. Partition: Default 4MB with spiffs, 업로드
3. 우산 1번을 대여
4. **ESP32 USB를 뽑았다가 다시 꽂기** (전원 재시작)
5. 시리얼에 `복원: 우산 1번은 김OO 가 빌린 상태` 가 뜨면 성공! 🎉
6. 우산 1번을 찍으면 정상적으로 반납됨

## 확인 ✅
- [ ] 대여 후 전원 재시작해도 빌린 상태가 복원된다
- [ ] 반납하면 상태가 비워지고, 그것도 저장된다
- [ ] `/log.csv` 에 대여/반납 내역이 쌓인다 (4단계에서 확인)

## 배운 것
- 상태를 파일에 저장/복원 = "꺼져도 잊지 않기"
- 덮어쓰기 저장(`FILE_WRITE`) vs 이어쓰기 기록(`FILE_APPEND`)의 차이
- 현재 상태(state)와 누적 기록(log)을 둘 다 관리
