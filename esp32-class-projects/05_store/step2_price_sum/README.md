# 2단계 — 상품 인식 + 가격 합산

## 목표
등록한 카드를 찍으면 상품으로 인식해 **가격을 더하고**, "결제" 카드를 찍으면 **합계를 보여주고 초기화**한다.

## 원리 — UID를 상품과 연결 (표 찾기)
1단계에서 읽은 UID를 **상품표**에 넣어두고, 찍힌 UID가 표에 있으면 그 가격을 더해요.
```cpp
String productUID[]   = { "A1B2C3D4", "11223344", "55667788" };
String productName[]  = { "초코파이",   "콜라",      "사탕"     };
int    productPrice[] = { 500,        1000,       300       };
```
찍힌 UID를 표와 하나씩 비교(`for` + `if`)해서 일치하면 가격 합산.

## 해보기
1. `step2_price_sum.ino` 열기
2. **상품표의 UID를 1단계에서 적은 내 카드 UID로 수정**
   ```cpp
   String productUID[PRODUCT_COUNT] = { "내UID1", "내UID2", "내UID3" };
   String CHECKOUT_UID = "내결제카드UID";
   ```
3. 이름·가격도 원하는 대로 수정
4. 업로드 → 시리얼 모니터
5. 상품 카드들 찍기 → 합계가 쌓이는지
6. 결제 카드 찍기 → 합계 표시 후 0으로 초기화

## 확인 ✅
- [ ] 상품 카드 찍으면 `이름 +가격원 (합계 ...)` 이 뜬다
- [ ] 여러 개 찍으면 합계가 누적된다
- [ ] 결제 카드 찍으면 `===== 합계: ...원 =====` 후 초기화
- [ ] 등록 안 한 카드는 "등록되지 않은 카드"로 표시

## 직접 바꿔보기
- 상품 개수 늘리기 (`PRODUCT_COUNT` 와 표 3줄 같이 수정)
- 부저 추가해서 찍을 때 "삑" 소리

## 배운 것
- UID ↔ 상품 정보를 **표(배열)** 로 연결
- `for` 로 표를 훑어 일치 항목 찾기
- 특별한 카드(결제)로 동작을 분기
