# AI Studio 프롬프트 원문 — 그대로 복사해서 시작

> `[ ]` 부분만 우리 팀 것으로 바꿔요 — 이 문장이 오늘의 '설계도'.
> 화면 구성 → 디자인 규칙 → 동작 규칙, **3단 구조**로 쓰면 실수가 없어요.

```
교실 환경 모니터 웹앱을 만들어줘.

화면 구성 :
- 상단 제목 : "[1팀] 교실 환경 모니터"
- 큰 카드 4개 : 온도, 습도, 조도, 소음
- 카드마다 큰 숫자와 단위 표시
- 하단 : LED 켜기 / LED 끄기 버튼 2개

디자인 규칙 :
- 파란 계열, 모바일 우선, 버튼 크게, 글자 크게

동작 규칙 :
- 10초마다 [우리 팀 웹앱 URL]?mode=latest 에서
  최신 값을 가져와 카드에 표시해줘
- LED 버튼을 누르면
  [우리 팀 웹앱 URL]?mode=set&cmd=ON (또는 OFF)
  으로 요청을 보내줘
```

## 체크리스트

- [ ] `mode=latest`·`set` 분기는 배포 템플릿 Apps Script([`03_apps_script_full.gs`](03_apps_script_full.gs))에 포함 — 이름만 짝 맞추면 동작
- [ ] URL 붙여넣기 전 끝이 **`/exec`** 인지 확인
- [ ] 생성 후엔 스케치·명세와 비교 → **한 번에 하나씩** 수정 지시
- [ ] 생성 결과는 팀마다 달라요 — 정답은 'AI 화면'이 아니라 '우리 스케치'!

## 심화 — 바깥 날씨 카드 추가 (선택)

```
Open-Meteo API(회원가입·키가 필요 없는 무료 날씨 API)를 사용해서,
위도 [37.57] 경도 [126.98] 위치의 현재 기온과 습도를 가져와
"바깥 날씨" 카드로 추가해줘.

교실 온도 카드와 나란히 배치해서 안팎이 비교되게 해줘.
10분마다 자동으로 갱신해줘.
```

- 위도·경도는 지도 앱에서 학교를 길게 눌러 확인 후 교체
- 키가 필요한 API(미세먼지 등)는 키를 발급받아 "이 키를 사용해줘 : [키]"라고 함께 주면 돼요
