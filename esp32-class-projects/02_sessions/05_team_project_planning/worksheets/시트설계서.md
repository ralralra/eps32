> **표의 열 제목을 정하는 것 = 데이터베이스 설계.**
> 설계 원칙: ① 기록(로그)은 지우지 않고 계속 쌓는다 ② 현재 상태와 기록을 다른 탭으로 분리한다

## 탭 구성

| 탭 이름 | 역할 (상태? 기록?) | 누가 쓰나 (웹앱? ESP32?) |
|---|---|---|
| 예: 상품 | 현재 상태 (재고) | 웹앱이 읽고 씀 |
| 예: 판매로그 | 기록 (계속 쌓음) | Apps Script가 씀 |
| 설정 | 명령 셀 (A1) | 웹앱이 쓰고 ESP32가 읽음 |

## ⚠ 이름 일치 점검 (4회차 '이름 대조표' 참고)

- [ ] 탭 이름 = Apps Script의 `getSheetByName("...")`
- [ ] 열 순서 = `appendRow([...])` 순서
- [ ] 웹앱 `action`·파라미터 이름 = Apps Script 분기 문자열
