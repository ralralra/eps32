# 1단계 — 카드 UID 읽기 (매점 1단계와 동일)

## 목표
학생증과 우산 태그의 UID를 각각 읽어 적어둔다.

## 해보기
1. RC522 배선 (매점과 동일, [`../docs/wiring_and_setup.md`](../docs/wiring_and_setup.md))
2. `step1_read_card.ino` 업로드 → 시리얼 모니터(115200)
3. 학생증들과 우산 태그들을 하나씩 대면서 UID 기록

## 확인 ✅
- [ ] 카드 댈 때마다 `UID: ########`
- [ ] 학생증 UID들과 우산 태그 UID들을 적었다

## 적어두기
```
학생 김OO = ______    우산 1번 = ______
학생 이OO = ______    우산 2번 = ______
```

## 배운 것
- 매점에서 배운 RFID UID 읽기가 그대로 재사용된다
- 같은 기술(UID)로 "사람"도 "물건"도 식별할 수 있다
