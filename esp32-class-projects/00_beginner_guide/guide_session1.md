# 🐣 1회차 왕초보 가이드 — LED · LCD · 첫 센서

> 준비물: Wemos D1 R32 + 고릴라 실드(결합된 상태) + USB 케이블 + LCD 셀 + 빛 셀
> 실드 결합: USB 방향 맞추고 수직으로 천천히! (핀 휘면 안 돼요)

---

## 실습 1. 내장 LED 깜빡이기 (첫 업로드)

**🎯 무엇을**: 보드의 파란 LED를 1초마다 깜빡이게 — 개발환경 합격 시험!

**🔌 배선**: **없음!** 보드 + USB만. (실드 안 껴도 됩니다)

**📄 코드**: [`01_blink.ino`](../02_sessions/01_esp32_first_meet/code/01_blink/01_blink.ino)

**👣 순서**
1. USB로 보드-컴퓨터 연결 → 보드에 빨간 전원 불 확인
2. Arduino IDE 실행 → `파일 → 새 스케치` → 원래 있던 글자 **전부 지우기**
3. 위 코드를 열어 **전체 복사** → IDE에 붙여넣기
4. `툴 → 보드 → esp32 → ESP32 Dev Module` 확인
5. `툴 → 포트 → COM○` — 안 보이면 케이블 뽑았다 다시 꽂고, **새로 생긴 번호** 선택
6. **화살표(→) 버튼** 클릭 (체크✓ 아님!) → 아래 검은 창에 주황 글자 진행

**✅ 성공이면**: 아래 창에 `Hard resetting...` + 보드의 **파란 불이 1초 간격으로 깜빡**

**🚨 안 되면**
- 포트가 아예 없음 → USB가 충전 전용 케이블인지 확인 → CH340 드라이버 설치
- `Connecting...`에서 멈춤 → 그 글자가 보일 때 보드의 **IO0(BOOT) 버튼을 2~3초** 꾹
- 업로드는 됐는데 안 깜빡임 → EN(리셋) 버튼 한 번

## 실습 2. 우리 팀 LED 신호

**🎯 무엇을**: 깜빡임 패턴 3가지(대기·작업중·완료)로 팀만의 신호 만들기

**🔌 배선**: 없음 (실습 1과 동일)

**📄 코드**: [`02_team_signal.ino`](../02_sessions/01_esp32_first_meet/code/02_team_signal/02_team_signal.ino)

**👣 순서**
1. 코드 전체 복사 → 붙여넣기 → 업로드
2. `delay(600)` 같은 숫자를 바꿔 보기 — 숫자가 작을수록 빨라져요 (예: 150)
3. 바꿀 때마다 **다시 업로드**해야 보드에 반영됩니다!

**✅ 성공이면**: 천천히 3번 → 빠르게 5번 → 길게 1번이 반복

## 실습 3. LCD에 글자 띄우기

**🎯 무엇을**: 작은 화면에 "Hello Team1!" 표시

**🔌 배선**: [LCD 연결 그림](../images/lcd_wiring_gorilla_shield.png) — 실드 **왼쪽 아래 I2C 블록**의 빈 포트에 4핀 케이블 (G→GND, V→VCC, 셋째 칸→SDA, 넷째 칸→SCL)

**📄 코드**: [`03_lcd_hello.ino`](../02_sessions/01_esp32_first_meet/code/03_lcd_hello/03_lcd_hello.ino)

**👣 순서**
1. (한 번만) `스케치 → 라이브러리 포함 → 라이브러리 관리` → `LiquidCrystal I2C` 검색 → 설치
2. 코드 복사 → 붙여넣기 → `"Hello Team1!"`을 우리 팀 이름으로 바꾸기 (**영문·숫자만!** 한글은 안 나와요, 한 줄 16자까지)
3. 업로드

**✅ 성공이면**: LCD 두 줄에 글자가 떠요

**🚨 안 되면**
- 화면이 켜지는데 글자가 없음 → 코드의 `0x27`을 `0x3F`로 바꿔 재업로드 → 그래도 안 되면 LCD 뒷면 **파란 다이얼**을 드라이버로 살짝 돌리기(대비 조절)
- `#include` 줄에서 에러 → 라이브러리 미설치 (1번으로)

## 실습 4. 빛센서 값을 LCD에 표시

**🎯 무엇을**: 센서 → 숫자 → 화면. IoT의 최소 단위 완성!

**🔌 배선**: LCD는 그대로 두고, [아날로그 셀 그림](../images/wiring_analog_cell.png)처럼 빛 셀을 **Analog 블록 A2~A5 위치**에 추가 (예: A3 자리)

**📄 코드**: [`05_light_lcd.ino`](../02_sessions/01_esp32_first_meet/code/05_light_lcd/05_light_lcd.ino)

**👣 순서**
1. 코드 복사 → 붙여넣기
2. `#define LIGHT 34` 확인 — A3 자리에 꽂았으면 그대로, 다른 자리면 바꾸기 (A2=35 · A4=36 · A5=39)
3. 업로드

**✅ 성공이면**: LCD에 `Light:` 뒤 숫자가 보이고, **손으로 센서를 가리면 숫자가 뚝 떨어져요!**

**🚨 안 되면**: 숫자가 0 또는 4095에 고정 → 셀이 A2~A5 위치인지 + 코드 번호가 그 자리 GPIO인지 확인

## 실습 5. 종합 미션 — 카운트다운 타이머

**🎯 무엇을**: LCD에 10→0 카운트다운, 0이 되면 `TIME OVER!` + LED 5번 점멸. (심화: 빛센서 가리면 일시정지)

**🔌 배선**: 실습 4와 동일 (LCD + 빛 셀)

**📄 코드**: [`10_lcd_countdown.ino`](../08_cell_samples/10_lcd_countdown/10_lcd_countdown.ino)

**👣 순서**
1. 코드 복사 → 붙여넣기 → 업로드
2. 다시 보고 싶으면 보드의 **EN(리셋) 버튼** — 처음부터 다시 시작돼요
3. 심화를 뺄 거면 `while (analogRead(LIGHT)...)` 부터 `lcd.print("     ");` 까지 지워도 OK

**✅ 성공이면**: 10부터 줄어들다 0에서 TIME OVER + 파란 불 깜빡깜빡. **영상으로 찍어서 팀 채널에!**
