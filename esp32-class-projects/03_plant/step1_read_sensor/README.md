# 1단계 — 토양 수분 센서 값 읽기

## 목표
센서가 흙의 촉촉함을 **숫자로** 알려준다는 걸 확인하고, 내 센서의 "마름/젖음" 값을 직접 잰다.

## 원리
정전용량식 센서는 흙에 물이 많을수록 전기 성질이 달라져요. ESP32의 **ADC(아날로그-디지털 변환)** 가 그걸 0~4095 숫자로 읽어줍니다.

```cpp
int raw = analogRead(SOIL_PIN);   // 0 ~ 4095
```

## 배선
센서 AOUT → **GPIO34**, VCC → 3V3, GND → GND
(자세히는 [`../docs/wiring_and_setup.md`](../docs/wiring_and_setup.md))

## 해보기
1. 배선 후 `step1_read_sensor.ino` 업로드
2. 시리얼 모니터(115200) 열기
3. 센서를 **공기 중**에 두고 값 확인 → 적어두기
4. 센서를 **물컵**에 담그고 값 확인 → 적어두기
5. 흙(마른 흙 / 젖은 흙)에서도 값이 다른지 보기

## 확인 ✅
- [ ] 시리얼에 `수분 센서 값: ####` 가 0.5초마다 나온다
- [ ] 공기 중 → 물컵으로 옮기면 **값이 뚜렷하게 변한다**
- [ ] 막대 그래프 `[####....]` 도 같이 변한다
- [ ] 공기 중 값과 물컵 값을 종이에 적었다

## 중요 — 내 센서 값 기록하기
센서마다 값이 달라요. 꼭 적어두세요:
```
공기 중(마름) 값 = ______   (큼)
물컵(젖음)   값 = ______   (작음)
마름 기준(임계값) ≈ 중간값 = ______   ← 3단계에서 씀
```

## 배운 것
- 센서 = 물리 현상을 숫자로 바꿔주는 부품
- `analogRead()` 로 0~4095 값을 읽는다
- 같은 센서라도 환경에 따라 값이 바뀐다 → **캘리브레이션(기준 정하기)** 이 필요
