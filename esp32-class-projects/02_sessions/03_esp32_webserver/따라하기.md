# 3회차 실습 따라하기 — 폰으로 보드 조종하기 (ESP32 웹서버)

> 준비물: 보드+실드, LCD 셀, DHT 셀, 스마트폰. **인터넷은 필요 없어요!**
> 오늘의 핵심 문장: **서버 = 요청을 받으면 응답하는 프로그램. 오늘부터 ESP32가 서버!**

---

## 실습 1. ESP32가 WiFi를 만들게 하기 (AP 모드)

**🎯 무엇을**: 보드가 스스로 `Team1_IoT`라는 WiFi를 만들어요

**🔌 배선**: 없음 (보드만)

**📄 코드**: [`01_ap_hello.ino`](code/01_ap_hello/01_ap_hello.ino)

**👣 순서**
1. 코드 복사 → 붙여넣기
2. `"Team1_IoT"`를 우리 팀 이름으로 (예: Team3_IoT)
3. 비밀번호는 **꼭 8자 이상!** (7자면 WiFi 자체가 안 만들어져요)
4. 업로드 → 몇 초 기다리기

**✅ 성공이면**: 아직 화면엔 없음 — 다음 실습에서 폰으로 확인!

---

## 실습 2. 스마트폰으로 접속

**👣 순서**
1. 폰 `설정 → WiFi` → 목록에서 **Team1_IoT** 찾기 (안 보이면 몇 초 뒤 다시 스캔)
2. 비밀번호 `12345678` 입력 → 연결
3. **"인터넷에 연결되어 있지 않음" 경고가 떠도 정상!** → "연결 유지" 선택
   (우리는 인터넷이 아니라 보드에 접속하는 거예요)
4. 폰 브라우저 주소창에 **`192.168.4.1`** 입력 (http:// 안 붙여도 됨)

**✅ 성공이면**: 화면에 **Hello ESP32!** — 내 폰이 내 보드의 웹페이지를 받아온 거예요! 옆 사람 폰으로도 접속해 보세요 (동시 접속 = 진짜 서버)

**🚨 안 되면**
- WiFi 목록에 없음 → 업로드 성공했는지 + 비밀번호 8자 이상인지 → 보드 EN(리셋) 버튼
- 연결은 되는데 페이지 안 열림 → 폰의 **모바일 데이터를 잠깐 끄기** (폰이 인터넷 되는 쪽으로 도망감)

---

## 실습 3. 폰 버튼으로 진짜 LED 켜기

**🎯 무엇을**: 지난주 "화면 글자만 바뀌던" 버튼이 드디어 **진짜 LED**를 켭니다

**🔌 배선**: 없음 (내장 LED 사용)

**📄 코드**: [`02_led_remote.ino`](code/02_led_remote/02_led_remote.ino)

**👣 순서**
1. 코드 복사 → 붙여넣기 → WiFi 이름만 팀 것으로 → 업로드
2. 폰 WiFi 재연결 → `192.168.4.1` 접속
3. [LED 켜기] 터치!

**✅ 성공이면**: 폰 터치 → **보드 파란 불 켜짐**. 방금 일어난 일: 터치 → fetch('/led/on') → ESP32 접수 → digitalWrite → 점등 (0.1초의 여행!)

**🚨 버튼 눌러도 안 켜지면**: 폰 주소창에 `192.168.4.1/led/on` 을 **직접** 입력해 보기
→ 이걸로 켜지면 웹페이지(JS) 문제 / 이것도 안 되면 코드(server.on) 문제 — 범인이 갈려요!

## 실습 4. 센서값을 폰 화면에 (미니 기상 스테이션)

**🔌 배선**: DHT 셀 → Digital 포트 ([그림](../../images/wiring_digital_cell.png)) — 코드의 `DHTPIN`을 꽂은 자리 GPIO로

**📄 코드**: [`03_sensor_json.ino`](code/03_sensor_json/03_sensor_json.ino)

**👣 순서**
1. 업로드 → 폰 재접속
2. **재료부터 검사**: 폰 주소창에 `192.168.4.1/sensor` → `{"temp":25.3,"humi":48.0}` 모양 확인
3. 그다음 `192.168.4.1` 접속 → 숫자가 2초마다 저절로 갱신!

**✅ 성공이면**: 센서에 입김 불면 몇 초 안에 폰의 습도 숫자가 확 올라요

## 실습 5. 종합 — 내 손안의 리모컨

**🎯 무엇을**: 한 화면에서 LED 켜고 끄기 + 글자를 교실 LCD로 보내기 + 온습도 실시간

**🔌 배선**: LCD([그림](../../images/lcd_wiring_gorilla_shield.png)) + DHT

**📄 코드**: [`04_full_remote.ino`](code/04_full_remote/04_full_remote.ino)

**👣 순서**
1. 업로드 → 폰 접속
2. 입력창에 영문으로 인사 쓰고 [보내기] → **교실 LCD에 그 글자가!**
3. 팀원 폰 2대로 동시 접속해서 서로 LCD에 인사 보내기 (하이라이트!)

**⚠ LCD는 한글이 안 나와요** — 영문·숫자로! (16자까지)
