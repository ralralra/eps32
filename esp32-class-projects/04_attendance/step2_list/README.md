# 2단계 — 버튼 누르면 명단에 쌓이게 (브라우저에서)

## 목표
출석 버튼을 누르면 **이름이 명단에 한 줄씩 추가** 되게 만든다. 아직 브라우저 안에서만.

## 왜?
1단계 버튼은 눌러도 아무 일이 없었죠. 이제 **버튼에 동작을 연결** 합니다.
이 "동작"을 적는 언어가 **자바스크립트(JavaScript)** 예요. `<script>` 안에 들어갑니다.

## 해보기
1. `index.html` 을 브라우저로 열기
2. 이름 입력 → "출석하기" 클릭
3. 아래 "출석한 사람" 목록에 `이름 - 시각` 이 추가되는지 확인
4. 여러 명 입력해서 계속 쌓이는지 확인

## 확인 ✅
- [ ] 이름 넣고 버튼 → 아래 목록에 한 줄 생김
- [ ] 시각도 같이 표시됨 (예: `홍길동 - 오후 2:15:30`)
- [ ] 입력칸을 비우고 누르면 "이름을 입력하세요" 경고가 뜸
- [ ] 여러 명 넣으면 줄이 계속 쌓임

## 코드에서 새로 생긴 것

```html
<button onclick="check()">출석하기</button>   ← 버튼 누르면 check() 실행
<ul id="list"></ul>                            ← 명단이 들어갈 빈 목록
```

```javascript
function check() {
  var name = document.getElementById("name").value;  // 입력칸 글자 읽기
  ...
  li.textContent = name + " - " + time;              // "이름 - 시각" 한 줄
  document.getElementById("list").appendChild(li);   // 목록에 추가
}
```

흐름: **버튼 클릭 → check() 실행 → 이름 읽기 → 목록에 추가**

## 직접 바꿔보기
- 시각 말고 "출석!" 같은 글자도 붙여보기
- 빈 이름일 때 경고 문구 바꿔보기

## ⚠ 한 가지 문제 발견하기
브라우저를 **새로고침(F5)** 해보세요. 명단이 전부 사라집니다!
왜? 지금은 화면에만 잠깐 보여줄 뿐, **어디에도 저장하지 않았기 때문**이에요.
→ 이 문제를 다음 단계에서 해결합니다.

## 배운 것
- 버튼에 `onclick` 으로 동작을 연결한다
- 자바스크립트로 입력값을 읽고 화면을 바꾼다
- 새로고침하면 사라진다 = **저장이 필요하다**
