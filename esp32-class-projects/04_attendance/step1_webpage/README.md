# 1단계 — 출석 웹페이지 만들기 (브라우저에서)

## 목표
출석 화면이 어떻게 생겼는지 **눈으로** 먼저 만든다. 아직 ESP32는 안 씀.

## 왜 이걸 먼저?
웹페이지는 그냥 **글로 적은 화면 설계도(HTML)** 입니다.
ESP32에 넣기 전에, 내 컴퓨터 브라우저에서 먼저 만들어 보면 "아, 이게 화면이구나" 가 바로 보여요.

## 해보기
1. `index.html` 파일을 더블클릭 → 크롬 같은 브라우저로 열기
2. 화면에 제목, 이름 입력칸, 출석 버튼이 보이는지 확인

## 확인 ✅
- [ ] "📋 우리 반 출석" 제목이 보인다
- [ ] 이름을 입력하는 칸이 있다
- [ ] 파란 "출석하기" 버튼이 보인다
- [ ] (아직) 버튼을 눌러도 아무 일도 안 일어난다 ← **정상!** 다음 단계에서 동작시킴

## HTML 뜯어보기

```html
<input id="name" placeholder="이름">   ← 이름 적는 칸
<button>출석하기</button>              ← 누르는 버튼
```

- `<h1>` = 큰 제목
- `<input>` = 입력칸 (`placeholder` = 비어있을 때 보이는 안내 글씨)
- `<button>` = 버튼
- `<style>` 안 = 색깔/크기 같은 꾸미기

## 직접 바꿔보기 (재미 + 이해)
- 제목 글자를 우리 반 이름으로 바꿔보기 (`<h1>` 안의 글자)
- 버튼 색을 바꿔보기 (`background: #2563eb` → `#16a34a` 초록 등)
- 저장하고 브라우저 새로고침(F5) → 바뀌었는지 확인

## 배운 것
- 웹페이지 = HTML 글로 만든 화면
- 브라우저로 `.html` 파일을 바로 열어서 확인할 수 있다
- 아직은 **보이기만** 하고 동작은 없다
