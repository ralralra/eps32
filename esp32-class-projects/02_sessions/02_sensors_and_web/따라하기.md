# 2회차 실습 따라하기 — 센서들 + 나의 첫 웹페이지

> 준비물: 1회차 세팅 + 진동·홀·소리·온습도(DHT)·초음파 셀, 그리고 메모장(또는 아무 텍스트 에디터)

---

## 실습 1. 시리얼 모니터로 센서 관찰 (오늘의 핵심 도구!)

**🎯 무엇을**: 보드가 컴퓨터로 보내는 숫자를 실시간으로 보기 — 앞으로 매일 쓰는 도구

**🔌 배선**: [디지털 셀 그림](../../images/wiring_digital_cell.png)처럼 진동 셀 → Digital 블록('6' 칸 포트), [아날로그 셀 그림](../../images/wiring_analog_cell.png)처럼 빛 셀 → A3 자리

**📄 코드**: [`01_vib_light_serial.ino`](code/01_vib_light_serial/01_vib_light_serial.ino)

**👣 순서**
1. 코드 복사 → 붙여넣기 → 업로드
2. IDE **우상단 돋보기 아이콘** 클릭 (= 시리얼 모니터)
3. 창 **우하단 속도를 115200 baud** 로 변경 ← 이거 안 하면 외계어 나와요!
4. 책상을 톡톡 두드리기 / 센서를 손으로 가리기

**✅ 성공이면**: `vib:0 light:3200` 같은 줄이 흐르고, 두드리면 vib가 1로 튀고, 가리면 light가 뚝

**🚨 안 되면**: 글자가 `⸮⸮⸮` 외계어 → 속도가 115200이 아님 (3번)

## 실습 2. 온습도(DHT) 읽기

**🔌 배선**: DHT 셀 → Digital 블록 포트 ([그림](../../images/wiring_digital_cell.png))
**📄 코드**: [`06_dht.ino`](../../08_cell_samples/06_dht/06_dht.ino) — `#define DHTPIN`을 꽂은 자리 GPIO로 (8→12 · 7→14 · 6→27 · 5→16)

**✅ 성공이면**: 시리얼에 `25.3C 48%` — **센서에 입김을 불면 습도가 확 올라요!**
**🚨 nan이 나오면**: 핀 번호 확인 + `delay(2000)` 그대로인지 (DHT는 2초에 1번만 측정 가능)

**🚨 항상 0이면**: TRIG·ECHO가 서로 바뀐 것 — 코드의 두 번호를 맞바꿔 재업로드

## 실습 5. 나의 첫 웹페이지 (HTML)

**🎯 무엇을**: 보드 없이, 메모장으로 웹페이지 만들기 — 다음 주에 이게 보드 속으로 들어가요!

**🔌 배선**: 없음 (컴퓨터만)

**📄 코드**: [`06_remote_step1.html`](code/06_remote_step1.html)

**👣 순서 (저장 방법이 절반!)**
1. 메모장 열기 → 코드 복사 → 붙여넣기
2. `파일 → 다른 이름으로 저장`
3. 파일 이름: `리모컨.html` / **파일 형식: "모든 파일"** ← 안 바꾸면 .txt가 붙어서 실패!
4. **인코딩: UTF-8** ← 안 하면 한글이 ???로 깨져요
5. 저장 위치: 바탕화면 팀 폴더 (다음 주에 또 씁니다!)
6. 저장한 파일을 **더블클릭** → 브라우저에 내 페이지가!

**✅ 성공이면**: 브라우저에 제목과 버튼 2개. 주소창이 `C:\...`로 시작하면 정상이에요 (아직 서버가 아니라 내 파일을 보는 중)

## 실습 6. CSS로 꾸미고, JS로 반응하게

**📄 코드**: 꾸미기 [`07_remote_step2_css.html`](code/07_remote_step2_css.html) → 동작 [`08_remote_step3_js.html`](code/08_remote_step3_js.html)

**👣 순서**
1. 메모장의 내용을 새 코드로 **전체 교체** → Ctrl+S 저장 → 브라우저에서 **F5**(새로고침)
2. 이 루틴을 외우세요: **수정 → Ctrl+S → F5**
3. step3까지 하면 버튼을 눌렀을 때 "LED 켜짐!" 글자가 바뀌어요

**⚠ 미리 알아두기**: 아직 **진짜 LED는 안 켜집니다!** 화면 글자만 바뀌는 게 정상 — 진짜로 켜지는 건 3회차에서

**🚨 버튼이 반응 없으면**: 브라우저에서 **F12 → Console 탭** → 빨간 에러 읽기. `ledOn is not defined` = 철자 틀림이 대부분
