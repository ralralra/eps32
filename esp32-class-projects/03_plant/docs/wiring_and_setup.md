# 화분 배선 & 설정 (Wemos D1 R32)

> 보드 처음 세팅: [`01_docs/setup_wemos_d1_r32.md`](../../01_docs/setup_wemos_d1_r32.md) ·
> 보드 라벨↔GPIO 대조: [`01_docs/pinmap_wemos_d1_r32.md`](../../01_docs/pinmap_wemos_d1_r32.md)
> D1 R32 보드에서 **GPIO34 = A3 자리**, **GPIO26 = D2 자리**입니다.
> 고릴라셀 토양습도 셀을 쓰면 배선 없이 **A2~A5 위치 포트**에 꽂기만 하면 돼요.

## 핀 연결

### 토양 수분 센서 (정전용량식)
| 센서 | ESP32 |
|---|---|
| VCC | 3V3 (또는 5V — 센서 사양 확인) |
| GND | GND |
| AOUT (아날로그 출력) | **GPIO34** |

> ⚠ **GPIO34** 를 쓰는 이유: ESP32의 ADC2(예: GPIO25,26 등)는 **와이파이를 켜면 아날로그 읽기가 안 됩니다.**
> 그래서 와이파이와 같이 쓰는 4단계를 위해 처음부터 **ADC1 핀(GPIO32~39)** 을 씁니다. GPIO34는 입력 전용이라 센서용으로 딱 좋아요.

### 릴레이 모듈
| 릴레이 | ESP32 |
|---|---|
| VCC | 5V (VIN) |
| GND | GND |
| IN | **GPIO26** |

### 워터펌프 (★ 외부 전원)
```
외부 5V(+) ──▶ 릴레이 COM
릴레이 NO  ──▶ 펌프(+)
펌프(-)    ──▶ 외부 5V(-) 와 ESP32 GND 공통 연결
```
- 펌프 전원은 **외부 5V**(충전기/건전지팩)에서. ESP32에서 직접 X
- 외부 전원의 GND와 ESP32의 GND는 **반드시 한 점에서 연결**(공통 그라운드)

## 보드 설정
- `Tools` → `Board` → **ESP32 Dev Module**
- `Tools` → `Upload Speed` → 115200
- (4단계만) 와이파이 사용 — 추가 설정 없음

## 센서 캘리브레이션 (1단계에서 함)
정전용량식 센서는 보통:
- **공기 중(마름)** → 값이 **큼** (예: 3000~4095)
- **물 속(젖음)** → 값이 **작음** (예: 1200~1600)

> 센서마다 값이 다르니, 1단계에서 **공기 중 값**과 **물컵 값**을 직접 적어두고,
> 그 중간쯤을 "마름 기준(임계값)"으로 정하세요. (3단계에서 사용)

## 릴레이 동작 방향 주의
릴레이 모듈은 두 종류가 있어요:
- **active-LOW**: `IN = LOW` 일 때 켜짐 (대부분)
- **active-HIGH**: `IN = HIGH` 일 때 켜짐

코드는 active-LOW 기준입니다. 펌프가 **반대로** 동작하면(가만히 있을 때 켜짐) 코드의 HIGH/LOW를 바꾸면 됩니다.

## 안전
- 물과 전기를 같이 다루니 **펌프/튜브 연결부 누수** 주의
- 첫 테스트는 **물컵 + 빈 화분**으로, 실제 식물은 동작 확인 후
- 펌프는 **물 없이 오래 돌리면 고장**나니, 급수 시간을 짧게(2초 등)
