# AI Thinker ESP32-CAM 핀맵

> 이 보드는 핀 대부분이 카메라/SD에 점유돼 있어, 외부 회로 붙일 때 충돌에 매우 주의해야 합니다.

## 카메라 전용 (건드리지 말 것)

| 신호 | GPIO |
|---|---|
| PWDN | 32 |
| XCLK | 0  ※ 부트모드 핀과 공유 |
| SIOD/SIOC | 26 / 27 |
| Y2..Y9 | 5, 18, 19, 21, 36, 39, 34, 35 |
| VSYNC/HREF/PCLK | 25 / 23 / 22 |

## SD 카드 (보드 옆 슬롯, SD_MMC 1-bit/4-bit 모드)

| 신호 | GPIO | 비고 |
|---|---|---|
| CMD | 15 | |
| CLK | 14 | |
| D0 | 2 | 부트 시 floating 권장 — 풀업 X |
| D1 | 4 | **흰 플래시 LED와 공유!** SD 4-bit 쓰면 LED 못 씀 |
| D2 | 12 | 부트 strapping — 풀다운 필요 |
| D3 | 13 | |

## 자유롭게 쓸 수 있는 핀 (사실상)

| GPIO | 비고 |
|---|---|
| 1 (TX) | 시리얼 송신과 공유. 디버깅 안 쓰면 사용 가능 |
| 3 (RX) | 시리얼 수신과 공유 |
| 16 | UART2 RX (Programmer 안 쓸 때) |

> 사실상 카메라 동작 중에는 GPIO 1/3/16 정도만 안전하게 추가 사용 가능.

## LED

| 위치 | GPIO | 로직 |
|---|---|---|
| 뒷면 빨간 LED | 33 | active-LOW |
| 앞면 흰 플래시 | 4  | active-HIGH (PWM 가능) |
