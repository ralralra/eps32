# 4단계 — 우산 대여 현황 웹 (완성본) 🎉

## 목표
폰으로 접속하면 **우산별 상태(사용가능/대여중+누구)** 와 **미반납 개수**를 한눈에 본다.

## 핵심 — 또 합체!
출석에서 배운 웹서버 + 우산 RFID 대여/반납 + 상태 저장의 합체예요.
매점 4단계와 거의 같은 구조 — 보여주는 내용만 "재고/매출" → "대여 현황"으로 바뀐 것.

## 해보기
1. UID들을 내 카드값으로 수정
2. Partition: Default 4MB with spiffs, 업로드
3. 폰으로 `ESP32-Umbrella` 접속 (비번 `umbrella1234`) → http://192.168.4.1
4. 카드로 대여/반납을 하면서 폰 화면(3초 새로고침)이 바뀌는지 확인

## 확인 ✅
- [ ] 우산별 상태가 표로 보인다 (사용가능/대여중)
- [ ] 대여하면 "대여중 + 빌린 사람" 으로 바뀐다
- [ ] 미반납 개수가 맞다
- [ ] 전원 껐다 켜도 현황이 유지된다

## 잘 안 될 때
- 웹 접속/카드 인식 문제 → 출석·매점 트러블슈팅과 동일

## 배운 것 (우산 전체 정리)
1. RFID UID 읽기 (매점에서 배운 것 재사용)
2. 학생/우산 두 표로 구분, 대여/반납 토글
3. 상태를 저장해서 전원 꺼도 유지
4. 웹으로 현황 조회

## 확장 아이디어
- 미반납 학생에게 알림 (인터넷 환경이면)
- 빈번 이용자 통계 (log.csv 집계 — 매점 4단계 방식)
- 서보모터로 거치대 잠금/해제 데모 (기획서 항목)
- 대여 시각 기록 (RTC 모듈 추가 시)
